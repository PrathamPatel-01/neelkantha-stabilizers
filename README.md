# Neelkantha Stabilizers - Premium Power Solutions Website

A high-performance, responsive landing page built for **Neelkantha Stabilizers**, an Ahmedabad-based manufacturer of industrial and residential voltage stabilizers. 

## 🚀 Features

* **Dynamic Product Details:** Uses Vanilla JavaScript to swap views between the product gallery and dedicated technical specification pages without reloading.
* **Modern Industrial UI:** A sophisticated Navy and Gold theme utilizing the *Playfair Display* and *Inter* font families.
* **Fully Responsive:** Optimized for desktops, tablets, and mobile devices using CSS Grid and Flexbox.
* **Interactive Elements:**
    * Scroll-linked navigation active states.
    * Intersection Observer API for "reveal on scroll" animations.
    * Floating WhatsApp integration for instant lead generation.
    * Filterable product gallery (Air Cooled, Oil Cooled, 3-Phase).
* **SEO Optimized:** Includes meta tags, canonical URLs, and structured data (JSON-LD) for local business visibility.

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3 (Custom Variables/Properties)
* **Logic:** Vanilla JavaScript (ES6+)
* **Icons:** FontAwesome 6.5.1
* **Fonts:** Google Fonts API

## 📂 Project Structure

* `index.html` - Main structure and SEO metadata.
* `styles.css` - Custom styling, animations, and responsive breakpoints.
* `script.js` - UI logic, product filtering, and view switching.
* `/images` - Product assets and company branding.
